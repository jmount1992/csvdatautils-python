CSVDataRow Module

class csvdatautils.csvdatarow.CSVDataRow(data: list, fields: list, mappings: dict = {})[source]

A helper class to store a single row of data contained within a CSV, or CSV like, file.

__init__(data: list, fields: list, mappings: dict = {}) → None[source]

Initialises a CSVDataRow object where the fields argument defines the set of class attributes. The mappings argument allows getting of an attribute via multiple names.

Numeric data will be stored as floats, all other data will be stored as strings except for the strings ‘’ and ‘none’ (or any variation: ‘NONE’, etc.) which will be stored as None.

Parameters

  • data (list) – the values for the provided fields

  • fields (list) – the names for each field provided within the data

  • mappings (bool, optional) – the field/attribute mappings. Allows the retrieval of the same data value via mutliple names. Format is {‘mapped_name’: ‘<field_name>’}. Defaults to {}.

Raises

ValueError – if the length of the fields and data arguments are not equal.
